# Лабораторная работа №5  
## Работа с JSON в Node.js

### Цель работы
Научиться работать с данными в формате **JSON** с помощью **Node.js**:  
читать, фильтровать, сортировать и преобразовывать данные.

---

### Описание задания
В рамках лабораторной работы выполняется обработка нескольких JSON-файлов с данными пользователей, клиентов и цветов.  
Каждое задание реализует отдельный скрипт:

1. **Task01** — фильтрация пользователей по географической широте (`lat < 0`) и сортировка по названию города в обратном алфавитном порядке.
2. **Task02** — фильтрация клиентов по городу `Кунгур`, сортировка сначала по полу (женщины первыми), затем по возрасту (по убыванию), затем по имени (по алфавиту).
3. **Task03** — вывод списка всех цветов из JSON в виде массива строк.
4. **Task04** — вывод списка цветов с соответствующими RGB-кодами в формате объектов `{color, rgb}`.
5. **Task05** — преобразует RGB-значения цветов из data.js в HEX-формат, сортирует цвета по имени и сохраняет результат в файл hexColors.json.
