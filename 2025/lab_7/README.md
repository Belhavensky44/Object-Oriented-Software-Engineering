# Лабораторная работа №7  
## Парсинг статической HTML-страницы

### Цель работы
Научиться получать данные со статической веб-страницы и извлекать информацию
из HTML-документа с помощью библиотеки **cheerio**.

---

### Описание задания
В рамках лабораторной работы выполняется парсинг турнирной таблицы
со статической страницы сайта со спортивными соревнованиями.
Данные на странице загружаются сразу в HTML и не требуют выполнения JavaScript.

В качестве источника данных использована страница Википедии
с турнирной таблицей Английской Премьер-лиги.

---

### Используемые технологии
- **Node.js**
- **axios** — загрузка HTML-страницы
- **cheerio** — парсинг HTML
- **fs** — запись файлов

---

### Извлекаемые данные
Из турнирной таблицы извлекаются следующие столбцы:
- Место в таблице
- Название команды
- Количество матчей
- Количество побед
- Количество очков

---

### Результаты работы
Полученные данные сохраняются в двух форматах:
- `table.json` — структурированные данные в формате JSON
- `table.csv` — данные в формате CSV для работы в табличных редакторах
